Hiding and showing content

There will be cases in which you will need to selectively “show” and “hide” content by either removing it from the visual flow or preventing screen readers from reading it. We have settled on the following conventions for each use case:

Visual
browsers		Screen
readers		Solution
hidehideAdd the boolean hidden attribute attribute to dynamically hide content from all users. Remove the attribute to show it again.
showhideUse the aria-hidden="true" attribute to “hide” content from screen readers while keeping it visually perceivable.
hideshowUse the u-visually-hidden class to show content to screen readers only. This removes the content from the visual flow of the document but retains its legibility by screen readers.
